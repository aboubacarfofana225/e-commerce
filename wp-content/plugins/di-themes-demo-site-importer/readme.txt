Di Themes Demo Site Importer: One Click Demo Import for theme.

== Description ==

Di Themes Demo Site Importer plugin can be use to import demo website developed by Di Themes.

To import demo website, Open: Appearance > Import Demo Website and simply click on 'Import Demo Website' button. it takes some time so wait for 2-10 minutes (depends on web hosting service).

If you are using theme developed by Di Themes, then you will not need to select demo data files. and also Di Themes Demo Site Importer plugin will set menu location, home page, blog page etc according demo website.

== Installation ==

= Traditional Installation =
1. Go to: Plugin > Add New menu.
2. Here, search for 'Di Themes Demo Site Importer'
3. Now click on 'Install' and then 'Activate'

= Manual Installation =
<ol>
	<li>Download plugin zip file from : https://wordpress.org/plugins/di-themes-demo-site-importer/</li>
	<li>Open Dashboard of your WordPress website.</li>
	<li>Open: Plugins > Add New > Upload Plugin > Browse... (select downloaded zip file) > Install Now</li>
	<li>Now simply activate plugin.</li>
</ol>
